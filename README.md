# The Kibbutz - Frontend Assets

This repository contains the UI assets, component definitions, and color palettes for "The Kibbutz" application.

## Directory Structure
- **ui tomer/**: Contains UI screenshots and component metadata.
  - **web/**: Web interface assets and JSON definitions.
  - **mobile/**: Mobile interface assets and JSON definitions.
- **logo/**: Project logo files.
- **color_palette/**: Color scheme images and JSON definition.

## Components
Component definitions are stored as JSON files, including:
- Source image reference
- UI Component descriptions
- Screen dimensions
- Placeholder positioning data (x, y, w, h)
